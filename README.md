## Getting Started

Clone the repository:
```bash
git clone https://github.com/BraveRam/pixn.git
cd pixn
```

Install dependencies:
```bash
npm install
```

Run the development server:
```bash
npm start
# or
npm run dev
```

Visit [https://pixn.vercel.app](https://pixn.vercel.app) for the live site.
